What is JavaScript and the DOM?

TL;DR: JavaScript is a programming language used for web development, enabling interactivity and dynamic content on websites. The DOM (Document Object Model) is a representation of a webpage's structure in a tree-like format, manipulated by JavaScript to update and modify webpage content dynamically.

How does JavaScript relate to HTML and CSS?

Building a website with HTML/CSS/JS is like constructing a building

  • HTML is the foundation and structure.
  • CSS is the aesthetics and design.
  • JavaScript is the interactive and functional components that make your building (or website) not just visually pleasing but also practical and engaging.

What are functions, control flow and loops?

Functions: In JavaScript, functions are like predefined sets of instructions that can be reused to perform a specific task or operation. They are helpful because they promote code reusability, organization, and modularity. Each step in making a pizza is like a function, for example the first step preparing dough is not specific to pizza and can be used in other recipes such as making donuts or bagettes.

Control Flow: Control flow is like the step by step instructions you follow when making a pizza. It's the order in which you do things to get something done:

  1. Prepare Dough
  2. Add Sauce
  3. Add Cheese

    4. Loops: At this point we want to start adding a heap of toppings and because this is a repetitive procedure, we can do it using a loop, repeating the next steps (a, b, c) until we've got all the toppings we want.

    1. Choose a Topping
    2. Prepare the Topping
    3. Add the Topping

    Now that we have added all the toppings, the control flow continues with the rest of the process:

  4. Bake
  5. Slice
  6. Serve

What is the DOM (Document Object Model)?

The DOM is an interface for web documents. It represents the web page as a tree of objects, where each part of the web page (like HTML elements, attributes, and text) is represented as a node in this tree.

The DOM allows you to interact with web pages dynamically using JavaScript, allowing you to access, and change the content, structure, and style of a web page.

The code snippet below shows a simple example of using JavaScript to manipulate the DOM. 

        
01  <html>
02    <body>
03      <p id="myText">Click the button to change this text!</p>
04      <button onclick="changeText()">Change Text</button>
05
06      <script>
07          function changeText() {
08              const element = document.getElementById('myText');
09              element.innerHTML = 'Text has been changed!';
10          }
11      </script>
12    </body>
13  </html>
  • Line 03: is a tag with the id myText with some text we want to change.
  • Line 04: Adds a button that will run the changeText() function when clicked.
  • Line 08: Searches the DOM for an element that has id myText and saves a pointer variable.
  • Line 09: Uses the elements innerHTML method to edit the text.

What's the difference between accessing data in arrays and objects in JS?

Arrays and objects are data structures that have some similarities in that they can hold collections of data.

Arrays are good for storing lists of similar items because you can easily sort or filter them, whereas objects are better for storing mixed data that can have named properties, such as storing details about a person (name, address, phone number, etc), and then accessing individual values using the key without having to loop through every bit of data to find what you want.
