CSS: prefers-color-scheme

19 September 2023

Simple auto light/dark mode with CSS Media Queries

The prefers-color-scheme CSS media feature is used to detect if a user has requested light or dark color themes. A user indicates their preference through an operating system setting (e.g. light or dark mode) or a user agent setting.

So I set up some default css styles which would apply for dark mode, so that would be the default theme. Then using prefers-color-scheme, I override the dark theme with whatever elements I want to change for light mode, such as background and font colour. 


  /* Default is dark mode */
  body {
    background-color: black;
    color: white
  }

  /* Override if in light mode */
  @media (prefers-color-scheme: light) {
  body {
    background-color: white;
    color: black
  }
