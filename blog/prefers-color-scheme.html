CSS: prefers-color-scheme

19 September 2023

Simple auto light/dark mode with CSS Media Queries

The prefers-color-scheme CSS media feature is used to detect if a user has requested light or dark color themes. A user indicates their preference through an operating system setting (e.g. light or dark mode) or a user agent setting.

Step 1: Set Up Default CSS Styles for Dark Mode

To start, let's define our default CSS styles for dark mode. This will serve as the foundation for our design. You can set these styles in a separate CSS file or within the <style> tag in your HTML file. Here's a simple example: 

    
body {
  background-color: #1a1a1a; /* Dark background color */
  color: #ffffff; /* Light font color */
}

/* Additional styles for dark mode */
/* Add more styles specific to dark mode as needed */

Step 2: Override Styles for Light Mode Using prefers-color-scheme

Now that we have our default dark mode styles, we'll use the prefers-color-scheme media query to customize the appearance for light mode. This feature detects the user's system preference for light or dark color schemes and applies the appropriate styles accordingly. 

    
@media (prefers-color-scheme: light) {
  body {
    background-color: #ffffff; /* Light background color */
    color: #333333; /* Dark font color */
  }

  /* Additional styles for light mode */
  /* Add more styles specific to light mode as needed */
}
